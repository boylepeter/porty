Here are some of them

HTML

Hypertext Markup Language

HTML is the rough textual markup of a webpage or app.

CSS

Cascading Style Sheets

This is how you make a page look, everything from format to colors and some animations.

Javascript

ES6

Javascript is a foudational language for web pages and apps. It can control everything from data to visuals.

jQuery

jQuery is an open source JavaScript library designed to simplify working with HTML DOM or Document Object Model as well as CSS and AJAX.

Handlebars

Handlebars is a Javascript library used to create reusable templates.

Bootstrap

Bootstrap is an open source grid framework created by developers at Twitter

React

React is an open source Javascript library used to create a user interface out of frontend components. It was developed by a team at Facebook (Meta).

APIs

APIs or Application Programming Interface(s) are the middlemen of the internet and allow different programs to talk to eachother.

Restful APIs

Restful APIs are a type of API that follow a representational state transfer architecture.

Git

Git is an open source version control system, it has many tools including GitHub and GitBash which are used widely throughout the industry.

GitHub

GitHub is an internet hosting provider which integrates Git as a version control system.

Heroku

Heroku is a platform as a service (PaaS) which allows developers to build and launch applcations entirely in the cloud.

Node

Node is a backend javascript runtime environment which executes code outside of your browser.

Express

Express is a back end web application framework for Node.js and has been called the go to server framework for Node

MySQL

Often pronounced "My Sequel"

MySQL is a relational database management system or RDBMS based on a structured query language or SQL.

MongoDB

MongoDB is a document oriented database program. It is often referred to as a NoSQL database.

Mongoose

Mongoose is a JavaScript object-oriented programming library that creates a connection between MongoDB and the Express web application framework

Sequelize

Sequelize is a promise-based Node.js object relational mapping (ORM) tool
